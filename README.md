# EmotionsClassifierApp

## How to run
1. *git clone https://github.com/Stanislav-3/EmotionsClassifierApp*
2. *cd EmotionsClassifierApp*
3. *make up*
4. Your page is successfully running on **127.0.0.1:80**

## How to stop
1. *make down*
